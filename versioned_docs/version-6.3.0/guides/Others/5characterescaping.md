**This page is about character escaping.**

Character escaping can be useful especially when using the embed parser.

---

| CHARACTER  | ESCAPED           | ALTERNATIVE |
|------------|-------------------|-------------|
| `[`        | `#RIGHT#`         | `\[`        |
| `]`        | `#LEFT#`          | `\]`        |
| `;`        | `#SEMI#`          | `\;`        |
| `:`        | `#COLON#`         | `\:`        |
| `$`        | `#CHAR#`          | `\$`        |
| `<`        | `#RIGHT_CLICK#`   | `\<`        |
| `>`        | `#LEFT_CLICK#`    | `\>`        |
| `=`        | `#EQUAL#`         | `\=`        |
| `{`        | `#RIGHT_BRACKET#` | `\{`        |
| `}`        | `#LEFT_BRACKET#`  | `\}`        |
| `,`        | `#COMMA#`         |             |
| `&&`       | `#AND#`           | `\&\&`      |
| `\|\|`     | `#OR#`            | `\\|\\|`    |
| `NEW LINE` |                   | `\n`        |

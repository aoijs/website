`$reactionData` holds data of a reaction.

## Usage

```php
$reactionData[option]
```

## Parameters

| Field  | Type   | Description                                                                                            | Required |
| ------ | ------ | ------------------------------------------------------------------------------------------------------ |:--------:|
| option | string | Option to retrieve. <br /> 1. **usernames** <br /> 2. **userIds** <br /> 3. **tags** |   true   |

$usersInChannel

$usersInChannel devolverá todos los usuarios conectados al canal de voz especificado.

Uso​

$usersInChannel[servidorID;opcion?;sep?]

Parámetros​

CampoTipoParámetrosRequerido
canalIDenteroCanal de voz.verdadero
opción?cadenaCómo devolver los usuarios
1. id (por defecto)
2. usuario - menciona a los usuarios		falso
sep?cadenaEl separador para separar los valores devueltos.falso

Ejemplo(s)​

Esto devolverá los usuarios conectados a un canal de voz:

bot.command({
    name: 'usersInChannel',
    code: `
  $usersInChannel[$voiceID;user;, ]
  `
});

---
title: onMessageDeleteBulk
description: This event triggers whenever a message is bulk-deleted within a guild.
id: messageDeleteBulk
---

import Example from "../../../../components/Page/Example.astro";
import { Aside } from '@astrojs/starlight/components';

<Aside title="Intents">

This requires the `GuildMessages` intent!

</Aside>

<Aside title="Functions" type="tip">

This event has a function to retrieve data:

[$bulk[option]](/functions/bulk)

</Aside>

<Example>

<div slot="default">

```js
client.messageDeleteBulkCommand({
    name: "Message Delete Bulk",
    channel: "$channelId",
    code: `
        $log[A lot of messages were just deleted!: $bulk[messages]]
    `
});
```

</div>
<div slot="handler">

```js
module.exports = {
    name: "Message Delete Bulk",
    channel: "$channelId",
    type: "messageDeleteBulk",
    code: `
        $log[A lot of messages were just deleted!: $bulk[messages]]
    `
};
```

</div>
</Example>

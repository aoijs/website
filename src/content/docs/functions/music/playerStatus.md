:::tip[Extension]

This requires aoi.music installed.

:::

`$playerStatus` will return the current Player Status.

## Usage

```php
$playerStatus
```

## Example(s)

This will return the current player status:

```javascript
client.command({
  name: "playerStatus",
  code: `
    $playerStatus
  `,
});
```

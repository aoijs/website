title: $stopPlayer

id: stopPlayer
---

:::tip[Extension]

This requires aoi.music installed.

:::

`$stopPlayer` will destroy the current player.

## Usage

```php
$stopPlayer
```

## Example(s)

This will destroy the current player:

```javascript
client.command({
  name: "stopPlayer",
  code: `
    $stopPlayer
  `,
});
```

:::tip[Extension]

`$shuffleQueue` will shuffle the current queue.

## Usage

```php
$shuffleQueue
```

## Example(s)

This will shuffle the current queue:

```javascript
client.command({
  name: "shuffleQueue",
  code: `
    $shuffleQueue
  `,
});
```

`$messageID` will return the message ID.

## Usage

```php
$messageID
```

## Example(s)

This will return the message ID of the message with which you executed the command:

```javascript
client.command({
  name: "messageID",
  code: `
  The message ID of your the message you just sent is: "$messageID"
  `
});
```

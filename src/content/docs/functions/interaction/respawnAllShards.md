`$respawnAllShards` will respawn all shards. (requires sharding)

## Usage

```php
$respawnAllShards
```

## Example(s)

This will respawn all shards (requires sharding):

```javascript
client.command({
  name: "respawnAllShards",
  code: `
   $respawnAllShards
  `
});
```

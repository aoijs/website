`$authorBannerColor` will return the profile banner color of the command author.

## Usage

```php
$authorBannerColor
```

## Example(s)

This will return your profile banner color (if any):

```javascript
client.command({
  name: "userBanner",
  code: `
  $authorBannerColor
  `
});
```

`$argsCount` will return the amount of given arguments.

## Usage

```php
$argsCount
```

## Example(s)

This will return the amount of arguments in your message, for example, `[prefix]argsCount Hello Bye` will return two:

```javascript
client.command({
  name: "argsCount",
  code: `
  $argsCount
  `
});
```

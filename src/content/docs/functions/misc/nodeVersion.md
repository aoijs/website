`$nodeVersion` will return your current node.js version.

## Usage

```php
$nodeVersion
```

## Example(s)

This will return the current node.js version your bot is running on:

```javascript
client.command({
  name: "nodeVersion",
  code: `
    $nodeVersion
    `
});
```

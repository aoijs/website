`$guildCount` will return the amount of guilds where your bot is in.

## Usage

```php
$guildCount
```

## Example(s)

This will return the amount of guilds your bot is in:

```javascript
client.command({
  name: "guildCount",
  code: `
  I'm in $guildCount guilds!
  `
});
```

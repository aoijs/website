`$abs` returns the absolute value of the number.

## Usage

```aoi
$abs[number]
```

## Parameters

| Field  | Type                                                                                              | Description | Required |
| ------ | ------------------------------------------------------------------------------------------------- | ----------- | :------: |
| number | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | The number. |   true   |

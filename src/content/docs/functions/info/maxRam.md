`$maxRam` will return the bot's maximum amount of Ram.

## Usage

```php
$maxRam
```

## Example(s)

This will return your bot's maximum amount of available Ram:

```javascript
client.command({
  name: "maxRam",
  code: `
  $maxRam
  `
});
```

`$datestamp` will return the day in MS.

## Usage

```php
$datestamp
```

## Example(s)

This will return day of the month, in MS ( we're using $humanizeMS to make it readable ):

```javascript
client.command({
  name: "datestamp",
  code: `
  $humanizeMS[$datestamp]
  `
});
```

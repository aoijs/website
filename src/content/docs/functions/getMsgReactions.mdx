`$getMsgReactions` will fetch and return the reactions of specified message
## Usage
```aoi
$getMsgReactions[messageID;sep?]
```
## Parameters
| Field   | Type                                                                                                                                                                                                 | Description                                            | Required |
| ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ | :------: |
| messageID | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number)                                                                                                    | The ID of the guild.                                   |   true   |
| sep? | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)                                                                                                    | The separator between each booster.                    |  false   |

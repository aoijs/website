:::tip[Extension]

This requires aoi.invite installed.

:::

`$inviteMaxUses` returns the maximum uses of the invite.

## Usage

```php
$inviteMaxUses
```

## Example(s)

This returns the maximum uses of the invite.

```javascript
$inviteMaxUses;
```

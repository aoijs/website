:::tip[Extension]

This requires aoi.invite installed.

:::

`$inviteUserID` returns the ID of the user that created the invite.

## Usage

```php
$inviteUserID
```

## Example(s)

This returns the ID of the user that created the invite.

```javascript
$inviteUserID;
```

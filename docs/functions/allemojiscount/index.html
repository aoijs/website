$allEmojisCount

With $allEmojisCount, you can see total of emojis on all servers as all, or with types

Usage#

$allEmojisCount[type?]

Fields#

FieldDescriptionTypeRequired
typeIncludes all type when didn't addstringno

Emoji Types#

  • [ ] â€” Returns all emojis count when left empty.
  • animated â€” Return all animated emojis count.
  • normal â€” Return all normal type emojis count.
  • roles â€” Return all emojis that specific roles only count.

Examples#

  • Without type:
bot.command({
name: "all-emojis-count",
code: `
$allEmojisCount amounts of emojis
`
//Returns: X amounts of emojis
});
  • With type:
bot.command({
name: "all-emojis-count",
code: `
$allEmojisCount[animated] animated emojis
`
//Returns: X animated emojis
});
